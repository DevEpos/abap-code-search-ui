Regular Expressions

To enable more complex search queries in the ABAP Code Search you can enable the regular expression in the search dialog.

Depending on the Release of the targeted ABAP system one of the following regular expression engines will be used

Release Engine
< 7.55 POSIX Regular Expressions
≥ 7.55 PCRE (Perl compatible regular expressions)
INFO
If you are restricted to the use of the POSIX regular expressions you may sometime experience errors during the search when the entered expression is too complex for the engine.
TIP
For more information about PCRE see
  • Short Syntax Overview
  • Perl documentation perlre
