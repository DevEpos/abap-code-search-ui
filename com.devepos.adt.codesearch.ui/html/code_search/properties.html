PCRE (Pearl compatible regular expressions) specific options

NOTE
PCRE is only available in systems higher or equal to 7.55. In older versions the older POSIX library will be used
  • Disable Extended Mode
    on a default basis the extended mode is activated and you have to escape the white spaces in your pattern. If you donÂ´t want to do that you activate this option and the spaces entered in the pattern are considered as they are
  • Enable Single Line Mode
    If this option is enabled the "." will also match the new line character during the search.

Parallel Processing

To reduce the search duration, you can enable parallel processing

  • Enable Parallel Processing
    by activating this checkbox the system will execute the search with parallel processes. Depending on the workload of the system the search will occupy a maximum of 10 processes if available.
  • Objects to search in a single API call
    This setting controls how many objects will be searched during a single back end call in an ongoing search query.
  • Server Group
    you can choose an existing server group which will be considered for the parallel processing. A validation of the entered value is available.
